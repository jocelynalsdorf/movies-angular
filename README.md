#Movie Reviews in Angular JS 
<h3>Author:</h3>
Jocelyn Alsdorf

<h3>Description:</h3>
This app allows a user to enter a movie, review it, see all reviews and get the films basic info via an api call.

<h2>Dependencies:</h2>

Angular JS v1.4.6
Angular ui-router
Bootstrap
